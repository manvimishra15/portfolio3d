VR Phobia Simulation

VR Phobia Simulation is an immersive virtual reality system designed to help people overcome common fears such as heights, darkness, and confined spaces through controlled exposure therapy. The project recreates realistic environments where users can gradually face their fears in a safe and controlled digital setting.

The system allows gradual exposure levels, making it suitable for therapeutic training or guided psychological therapy. It demonstrates how VR technology can be used to support mental health treatments.

Tech Stack: Unity (VR Development), C#, 3D Environment Design

ParkEase

ParkEase is a smart parking management system that helps users locate available parking spots quickly in crowded areas. The project aims to reduce traffic congestion caused by vehicles searching for parking.

The system displays parking slot availability and provides a visual interface for drivers to identify free spaces efficiently. It also demonstrates how digital infrastructure can improve urban mobility.

Tech Stack: HTML, CSS, JavaScript, Database Integration (Backend), API Integration

Greenify

Greenify is an environmental sustainability platform designed to promote eco-friendly habits and increase awareness about environmental conservation. The platform encourages users to adopt greener lifestyles and track their positive environmental impact.

The system can track sustainable activities such as reducing plastic use, recycling, and energy conservation. It demonstrates how digital platforms can motivate individuals to contribute toward environmental sustainability.

Tech Stack: Unity, Database Integration, Web Technologies

Crowd Management System

The Crowd Management System is designed to monitor and analyze crowd density in public spaces such as stadiums, festivals, and transport hubs. The goal is to identify overcrowded zones and prevent dangerous situations such as stampedes.

The system analyzes crowd data and movement patterns to detect risk levels and generate alerts when crowd density exceeds safe limits. This helps authorities manage large gatherings more efficiently.

Tech Stack: Python (Backend Processing), Computer Vision Concepts, Data Monitoring Interface

Dementia Support System

The Dementia Support System is designed to assist dementia patients in managing daily routines and remembering important tasks. The system provides reminders for medication, appointments, and daily activities.

It also supports caregivers by allowing them to monitor patient routines and ensure their safety. The goal is to improve independence and quality of life for people suffering from memory disorders.

Tech Stack: Web Technologies, Reminder System, Database Support
